# 🚀 国内免费部署指南

## 方案一：Gitee Pages（推荐，国内最稳定）

### 步骤

1. **注册 Gitee 账号**
   - 访问 https://gitee.com 注册

2. **创建新仓库**
   - 点击右上角 "+" → 新建仓库
   - 仓库名：`portfolio`
   - 选择：公开
   - 勾选：使用 Readme 文件初始化仓库

3. **推送代码**
   ```bash
   cd d:\UnityProject\portfolio
   git remote add origin https://gitee.com/你的用户名/portfolio.git
   git push -u origin master
   ```

4. **开启 Gitee Pages**
   - 进入仓库 → 服务 → Gitee Pages
   - 选择分支：master
   - 点击启动

5. **访问网站**
   - 地址：https://你的用户名.gitee.io/portfolio

**优点**：国内访问快，稳定，免费
**缺点**：需要审核（通常1-2天）

---

## 方案二：Vercel（最快，自动部署）

### 步骤

1. **注册 Vercel 账号**
   - 访问 https://vercel.com 注册（可用GitHub登录）

2. **安装 Vercel CLI**
   ```bash
   npm install -g vercel
   ```

3. **登录并部署**
   ```bash
   cd d:\UnityProject\portfolio
   vercel login
   vercel --prod
   ```

4. **按提示操作**
   - Project name: portfolio
   - Directory: ./ （当前目录）

5. **访问网站**
   - 地址：https://portfolio.vercel.app

**优点**：部署快（1分钟），自动HTTPS，全球CDN
**缺点**：国内访问可能稍慢

---

## 方案三：Netlify（简单）

### 步骤

1. **注册 Netlify 账号**
   - 访问 https://netlify.com 注册

2. **直接拖拽部署**
   - 登录后点击 "Add new site"
   - 选择 "Deploy manually"
   - 拖拽 `portfolio` 文件夹到页面

3. **访问网站**
   - 地址：https://xxx.netlify.app

**优点**：最简单，拖拽即可
**缺点**：国内访问可能稍慢

---

## 方案四：Coding Pages（腾讯云，国内快）

### 步骤

1. **注册腾讯云账号**
   - 访问 https://cloud.tencent.com 注册

2. **创建 Coding 项目**
   - 进入控制台 → Coding → 创建项目
   - 项目名：portfolio

3. **推送代码**
   ```bash
   cd d:\UnityProject\portfolio
   git remote add origin https://coding.tencent.com/你的用户名/portfolio.git
   git push -u origin master
   ```

4. **开启静态网站托管**
   - 进入项目 → 服务 → 静态网站
   - 选择分支：master
   - 点击保存

5. **访问网站**
   - 地址：https://你的用户名.coding.net/public/portfolio/overview

**优点**：国内访问快，稳定
**缺点**：需要实名认证
